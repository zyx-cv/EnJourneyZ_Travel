# EnJourneyZ

Travel guide HTML library.

## Structure

- `index.html`: homepage for all travel guides.
- `indonesia/index.html`: Indonesia Golden Week guide.

## GitHub Pages

Recommended repository name: `EnJourneyZ`.

After pushing this folder to GitHub:

1. Open repository `Settings`.
2. Go to `Pages`.
3. Set source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Visit `https://<github-username>.github.io/EnJourneyZ/`.

## Add A New Trip

Create a new folder such as `japan/` or `italy/`, put that trip's `index.html` inside it, then add a card to the root `index.html`.
