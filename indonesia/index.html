Before Departure

基础攻略

建议出发前 2 周开始处理签证、船票、包车和火山/出海确认。

证件与入境

  • 中国护照可申请 Indonesia eVisa / Visitor Visa，也可按官方口岸规则办理落地签；官方页面列出中国在适用国家中。
  • 印尼 Visitor Visa 为单次入境，30 天停留，官方费用 IDR 500,000；护照需自抵达日起至少 6 个月有效。
  • 建议提前 48 小时以上申请 eVOA / Visitor Visa，保存电子版与打印版。
  • 入境前填写 All Indonesia Arrival Card，准备护照、航班、酒店地址、签证信息。
  • 进入巴厘岛需支付 Love Bali 外国游客 Levy，官方 FAQ 为 IDR 150,000 / 人 / 次，保存 QR voucher。

钱、通讯、支付

  • 现金：准备 IDR 小额现金，火山、瀑布、司机小费、停车费、码头费和临时厕所常用现金。
  • 支付：Visa / Mastercard 在酒店和部分餐厅可用；小店、码头、乡村景点不要假设能刷卡。
  • 换汇：机场少量应急，主力在泗水/巴厘岛正规 money changer；避免路边超高汇率点。
  • 通讯：优先 Telkomsel 或 XL，覆盖东爪哇和巴厘岛更稳；到 Labuan Bajo 及海上会有信号断点。
  • 离线地图：Google Maps 区域离线包 + Maps.me / Organic Maps 备用。

关键风险

  • 布罗莫凌晨非常冷，火山口沙尘大；口罩和保暖层别省。
  • Tumpak Sewu 下谷底湿滑，手机防水袋和防滑鞋是刚需。
  • 科莫多出海受风浪影响，提前一天和船公司确认集合点、船型、是否含门票。
  • Angel's Billabong 只在低潮、海况稳定时远观或短暂停留；涨潮和大浪不要下水。
  • Nusa Penida 西线道路窄且颠，晕车药、包车司机和码头时间要留余量。
Climate & Outfit

天气与穿搭

9 月底到 10 月初处在旱季尾声/雨季开端，海岛与山区体感差异很大。

天气判断

Java、Bali、Komodo 所在的印尼西部和中部岛屿通常 5-9 月为旱季，10 月起逐渐进入雨季；全年白天平均约 28°C。你的行程横跨 9 月 26 日到 10 月 5 日，整体仍适合火山、海岛和出海，但要按“晴热 + 午后短阵雨 + 山区夜冷”的组合准备。

东爪哇高海拔区，如 Bromo，夜间和日出前会明显降温；海边和 Labuan Bajo / Bali 白天暴晒，紫外线强，船上风大但容易晒伤。

白天 27-31°C Bromo 凌晨约 8-14°C 海上防晒强度高 10 月需带轻雨具

分场景穿搭

  • 飞行与中转：短袖 + 薄长裤 + 轻外套，机舱和机场冷气足。
  • Tumpak Sewu：速干短袖/长袖、防滑涉水鞋或抓地好的溯溪鞋、可湿的短裤，背包加防雨罩。
  • Bromo：抓绒或薄羽绒、冲锋衣/防风外套、长裤、帽子、口罩；看完日出再逐步减衣。
  • Komodo 出海：泳衣内穿、长袖防晒衣、速干短裤、遮阳帽、偏光太阳镜、轻便拖鞋/溯溪鞋。
  • Bali 寺庙：有袖上衣，膝盖以下下装；Tanah Lot 可备围巾或 sarong。
Packing List

装备清单

按“火山、瀑布、海岛、航班转场”四种场景打包。

必带

  • 护照原件、签证/eVOA 打印件、返程机票、酒店订单
  • Love Bali Levy QR、All Indonesia Arrival Card QR
  • 境外保险单、紧急联系人纸质备份
  • Visa / Mastercard、少量美元或人民币备用、IDR 现金
  • Type C/F 转换插头，支持 230V 的充电器
  • 充电宝、数据线、相机电池、插线板
  • 防晒霜 SPF50、晒后修复、墨镜、帽子
  • 驱蚊液、止痒、创可贴、碘伏棉签
  • 肠胃药、晕车/晕船药、感冒药、止痛药
  • 防水手机袋、干湿分离袋、轻量雨衣
  • 防滑涉水鞋、拖鞋、轻便运动鞋
  • 口罩/魔术头巾，布罗莫沙尘和火山灰备用

可选但很提升体验

  • 小型望远镜：Lovina 海豚和 Komodo 海面观察有用。
  • GoPro / 防水相机：瀑布水汽和出海浮潜环境比手机安全。
  • 折叠水杯和电解质粉：火山日出、Nusa Penida 西线都容易脱水。
  • 轻便坐垫或薄毯：凌晨 Bromo 等日出和船上休息舒服很多。
  • 一次性马桶垫/湿巾：东爪哇路上补给点条件不稳定。
  • 小额防水零钱包：船费、小费、停车费、码头税分开放。

电压提醒：Indonesia uses Type C/F, 230V, 50Hz。大多数手机/电脑充电器支持 100-240V，只需转换插头；吹风机、卷发棒需检查铭牌。

Daily Playbook

每日作战卡

同一天的详细攻略、一步步执行、路书交通放在一起，旅途中不用上下翻找。

2026-09-26D1 抵达东爪哇

北京大兴 → 吉隆坡 → 泗水 → Lumajang, East Java。住宿 Indah Damai Cottages / 美丽和平小屋。

详细攻略

  • 航班 D7343 02:15-08:30，北京大兴到吉隆坡。
  • 航班 QZ323 15:30-17:10，吉隆坡到 Surabaya Juanda。
  • 抵达泗水后直接包车去 Lumajang，夜车以安全到达为第一目标。

一步步执行

  1. 00:00 到大兴，问清行李是否直挂 SUB。
  2. 08:30 到吉隆坡，先找 QZ323 登机口，再吃饭充电。
  3. 17:10 到泗水，入境后取现金/SIM，联系司机。
  4. 上车前拍车牌，发实时位置，到酒店马上睡觉。

路书交通

  • 机场：Beijing Daxing / Kuala Lumpur International / Surabaya Juanda。
  • 酒店地址：Jl. Nasional III No.54, Besukcukit, Sidomulyo, Lumajang。
  • 泗水到 Lumajang 约 3.5-5 小时，夜间只停正规加油站或便利店。
2026-09-27D2 Tumpak Sewu

Lumajang → Tumpak Sewu Waterfall → Teras Semeru → Cafe Lava Hostel。

详细攻略

  • Tumpak Sewu 07:00-13:00，先看观景台，再决定是否下谷底。
  • Teras Semeru 远观 Semeru 火山，火山活动以当地提示为准。
  • 晚上住 Cafe Lava Hostel，为 Bromo 凌晨日出做准备。

一步步执行

  1. 06:00 穿涉水鞋，手机进防水袋。
  2. 07:00 到瀑布，先拍全景，别急着下谷底。
  3. 08:00 问向导水量和安全；雨大就不下谷底。
  4. 13:00 换干衣服午餐，下午转场到 Bromo 山脚。

路书交通

  • Tumpak Sewu 地址：Jl. Raya Dampit-Lumajang, Sidomulyo, Lumajang。
  • Cafe Lava Hostel 地址：Jl. Raya Bromo No.RT.03, Cemorolawang, Ngadisari。
  • Lumajang 到 Bromo 山脚路长且弯，尽量天黑前抵达。
2026-09-28D3 Bromo + 飞科莫多

Mount Bromo 02:00-10:00；IU726 15:30-18:00 泗水 → Labuan Bajo。

详细攻略

  • 凌晨 Jeep 看 Bromo 日出，再去沙海和火山口。
  • 上午必须控制时间，下午赶 Surabaya Juanda 航班。
  • 抵达 Labuan Bajo 后入住 Wimolas Guesthouse。

一步步执行

  1. 01:30 起床，穿保暖层，证件随身。
  2. 02:00 Jeep 出发，提醒司机你要赶 IU726。
  3. 10:00 回酒店洗漱拿行李；晚于 10:30 就直接去机场。
  4. 18:00 抵达 Labuan Bajo，晚上确认 D4/D5 船。

路书交通

  • Bromo → Surabaya Airport 约 3.5-5 小时，堵车风险高。
  • 航班：IU726，Surabaya Juanda → Komodo International Airport。
  • Wimolas Guesthouse：Jl. Soekarno Hatta, Gorontalo, Komodo, West Manggarai。
2026-09-29D4 海钓恢复日

Labuan Bajo 海钓半日游，下午休整补给。

详细攻略

  • 半日海钓用于恢复体力，不要把当天塞满。
  • 下午采购 D5 出海用水、防晒、干粮和现金。
  • 继续住 Wimolas Guesthouse。

一步步执行

  1. 出海前问船家：接送、救生衣、水、午餐、钓具、取消政策。
  2. 上船前 30-60 分钟吃晕船药。
  3. 回酒店午睡，晚上确认 Komodo 船集合点和门票包含项。

路书交通

  • 酒店到 Labuan Bajo 码头通常不远，但清晨集合别卡点。
  • 所有船只以风浪为准，船长说取消就取消。
  • 现金零钱分开放，码头和小费常用。
2026-09-30D5 Komodo National Park

科莫多国家公园、巨蜥、Padar / Pink Beach 等出海点位视船线和天气而定。

详细攻略

  • Pitravel 原时间为 13:00-15:30，但常规一日出海通常更早。
  • 巨蜥必须跟 ranger，不能喂食、靠近或单独离队。
  • Padar 和 Pink Beach 是否执行取决于船型、风浪和套餐。

一步步执行

  1. 前一晚把泳衣、防晒衣、毛巾、防水袋、现金装好。
  2. 上船前拍船名，确认救生衣位置。
  3. 巨蜥徒步听 ranger 指挥；浮潜只在船员允许处下水。
  4. 返程后冲洗装备，确认 D6 QZ643。

路书交通

  • 出海口：Labuan Bajo Harbour。
  • 核心区域：Komodo National Park / Padar Island / Pink Beach。
  • 不要要求船长硬赶点位，风浪优先于打卡。
2026-10-01D6 飞巴厘岛

QZ643 15:10-16:15，Komodo International Airport → Ngurah Rai International Airport。

详细攻略

  • 上午不加远景点，只整理、洗衣、休息。
  • 抵达 Bali 后住 Rhadana Hotel。
  • 晚上确认 D7 Sanur 码头船票和 Nusa Penida 包车。

一步步执行

  1. 12:30 从酒店去 Komodo Airport。
  2. 15:10 飞 Bali，机上无餐食，提前买水。
  3. 16:15 抵达后用 Grab/Gojek 或酒店接送。
  4. 晚上补现金、防晒、水，早点睡。

路书交通

  • 航班：QZ643。
  • 酒店：Rhadana Hotel，Jl. Raya Kuta No.88, Kuta, Badung, Bali 80361。
  • 机场到 Kuta/Badung 约 20-60 分钟，堵车看时段。
2026-10-02D7 Nusa Penida 西线

Kelingking Beach 08:30-10:30；Broken Beach 11:00-12:00；Angel's Billabong 12:15-13:15。

详细攻略

  • Kelingking 观景为主，不建议一日游下到底部海滩。
  • Broken Beach 看天然拱桥和悬崖海湾。
  • Angel's Billabong 只在低潮且海况安全时靠近。

一步步执行

  1. 05:30 起床，穿可湿鞋，泳衣内穿。
  2. 06:00 去 Sanur Harbour，找船公司柜台。
  3. 08:30 到 Kelingking，只在安全平台拍照。
  4. 下午按返程船时间提前回码头，别错过船。

路书交通

  • 路线：Rhadana Hotel → Sanur Harbour → Nusa Penida → 西线包车。
  • 岛上路颠，晕车药和水必须带。
  • 返程后回 Rhadana Hotel，确认 D8 02:30-03:00 出发。
2026-10-03D8 Lovina + Ubud + Jimbaran

Lovina Dolphin Watching 05:30-08:00；Ubud Palace；Jimbaran Beach。

详细攻略

  • 这是全程第二高强度日，南部酒店往返 Lovina 很累。
  • Ubud Palace 只轻停留，不做深度乌布路线。
  • Jimbaran 日落海鲜需先确认重量、价格和税费。

一步步执行

  1. 02:30 上车去 Lovina，上车就睡。
  2. 05:30 上船看海豚，手机进防水袋。
  3. 如果太累，取消 Ubud，回酒店午睡。
  4. 17:00 到 Jimbaran，看清菜单再点。

路书交通

  • 路线：Kuta/Badung → Buleleng / Lovina → Gianyar / Ubud → Jimbaran。
  • 全天车程长，司机必须提前约好。
  • 回酒店后别加夜生活，D9 还要滑翔伞和 Tanah Lot。
2026-10-04D9 滑翔伞 + Tanah Lot

Tanah Lot 13:30-15:30；Krida Mandala I Ketut Lotri / Gunung Payung 一带滑翔伞。

详细攻略

  • 滑翔伞按风况执行，不要为了打卡硬飞。
  • Tanah Lot 最好改到日落前 60-90 分钟。
  • 最后一晚必须整理行李和返程随身包。

一步步执行

  1. 上午先问滑翔伞商家：风况、坐标、体重限制、保险。
  2. 滑翔伞不能飞就改 Uluwatu / Canggu 轻松线。
  3. Tanah Lot 不站湿滑礁石外侧。
  4. 晚上把证件、药品、充电宝、外套放随身包。

路书交通

  • 滑翔伞基地多在 Badung 南部 Gunung Payung / Kutuh。
  • Tanah Lot 在西北方向，两者不顺路，先按风况再去海神庙。
  • 回酒店后确认 D10 机场车。
2026-10-05D10 返程

Ngurah Rai International Airport 10:05 出发，新加坡中转 11 小时 45 分，06:35 抵达北京大兴。

详细攻略

  • 行李直达北京；新加坡停留时间长，但不要安排太满。
  • 如入境新加坡，提前填 SG Arrival Card。
  • 跨天航班注意当地时间和登机口。

一步步执行

  1. 06:30 从酒店去机场，国际航班按提前 3 小时处理。
  2. 值机问清行李直挂、下一程登机牌、登机口开放时间。
  3. 新加坡只选 Jewel Changi 或滨海湾短线。
  4. 登机前先过安检边检，再购物吃饭。

路书交通

  • 机场：Ngurah Rai International Airport → Singapore Transit → Beijing Daxing。
  • 随身包：护照、外套、充电器、药品、洗漱小包、银行卡现金。
  • 到北京后按入境、行李、海关、回家交通顺序走。
Day by Day

详细行程

以下按原始 Word 行程扩写，未写明的航班时刻以出票信息为准。

2026-09-26D1
抵达东爪哇

北京大兴 → 吉隆坡 → Surabaya 泗水 → Lumajang, East Java

北京大兴国际机场出发，经吉隆坡国际机场中转约 7 小时，行李直挂泗水。抵达 Surabaya Juanda International Airport 后建议提前约好包车，直接前往 Lumajang, East Java 区域入住。Lumajang 是东爪哇的县/地区，适合作为 Tumpak Sewu 瀑布的前后落脚点。

  • 中转：若不入境马来西亚，确认登机牌是否全程出齐、行李牌目的地为 SUB。
  • Surabaya 泗水到 Lumajang：约 3.5-5 小时，夜间抵达更建议包车，不建议临时找车。
  • 抵达后只做补给：水、早餐、现金、第二天瀑布门票零钱。

住宿

美丽和平小屋 / Peaceful Beautiful Lodge

当天目标

顺利入境、拿到现金/SIM、抵达 Lumajang 睡觉。

备选

若航班晚点严重，考虑在泗水机场附近过夜，D2 清晨再去 Tumpak Sewu。

2026-09-27D2
瀑布与火山

Tumpak Sewu 瀑布 → Semeru 视野 → Cemoro Lawang

Tumpak Sewu 是行程第一天的体力重点。先看观景台，再根据水量、天气和体力决定是否下到谷底。谷底路段会涉水、攀爬湿滑石阶和竹梯，建议请当地向导。

  • 建议节奏：07:00-08:00 到入口，先观景台拍全景；09:00 前开始下谷底。
  • 装备：涉水鞋、防水袋、速干衣、轻雨衣；相机镜头准备擦镜布。
  • 午后转场：从 Lumajang / Malang 方向去 Bromo 山脚 Cemoro Lawang，山路时间长，尽量天黑前抵达。
  • Semeru 火山如有火山灰雨或喷发提醒，以当地管理方和司机建议为准，不强行靠近。

住宿

熔岩咖啡旅馆 / Cafe Lava Hostel

重点

Tumpak Sewu 谷底路不适合拖鞋；雨后水量大时只看观景台也值得。

晚间准备

确认 Bromo Jeep 集合时间，通常 02:30-03:30；保暖衣物放在最外层。

2026-09-28D3
Bromo 日出

Mount Bromo → Surabaya Airport → Labuan Bajo

凌晨乘 Jeep 上观景点看 Bromo 火山群日出，之后下沙海，可骑马或步行到火山口楼梯。结束后回酒店拿行李，驱车前往泗水机场，飞往 Komodo Airport / Labuan Bajo。Labuan Bajo 属于 West Manggarai / Labuan Bajo 区域，是去 Komodo National Park 的主要出海口。

  • 日出点：Penanjakan / King Kong Hill 等，司机会按当日人流安排。
  • 火山口：沙海灰尘大，口罩、墨镜、帽子非常必要；楼梯陡，别靠近无护栏边缘。
  • 转机风险：Bromo 到泗水机场约 3.5-5 小时，至少给国内航班预留 3 小时机场缓冲。
  • 飞行：泗水朱安达国际机场 → 科莫多国际机场，机上无餐食，提前备简餐。

住宿

维莫拉斯宾馆 / Wimolas Guesthouse

当天体感

凌晨冷、上午晒、下午赶路，属于全程最累的一天。

抵达后

在 Labuan Bajo 码头街确认 D4 海钓和 D5 Komodo 出海集合点。

2026-09-29D4
海钓半日

Labuan Bajo 海钓半日游

前一天刚完成跨岛转场，这天安排半日海钓是合理的恢复日。建议上午出海，下午回镇上休整、洗衣、补给，并最终确认次日 Komodo National Park 船票。

  • 出海前确认：船型、救生衣、饮用水、钓具、是否接送酒店、是否含午餐。
  • 晕船：提前 30-60 分钟服药，空腹和过饱都不适合。
  • 下午：可去码头街、咖啡馆、日落点；不要把体力耗尽。
  • 采购：防晒、矿泉水、干粮、防水袋、现金，D5 岛上一切更贵且选择少。

住宿

维莫拉斯宾馆 / Wimolas Guesthouse

餐饮

Labuan Bajo 海鲜店多，注意选择明码标价、评价稳定的店。

2026-09-30D5
科莫多

Komodo National Park 一日出海

核心目标是在向导陪同下徒步寻找科莫多巨蜥，并尽量串联 Padar Island、Pink Beach、Manta Point / Taka Makassar 等海上点位。实际线路取决于船型、天气、风浪、园区规定和运营商套餐。

  • 集合：常见为 05:30-06:30 码头集合，快艇线路更早更紧凑。
  • 巨蜥：必须跟 ranger，不要喂食、不要靠近、不要单独离队；女性经期需主动告知向导。
  • Padar：台阶暴晒，带足水；清晨拍照更好，正午很热。
  • Pink Beach：可浮潜但注意珊瑚，穿水母衣/防晒衣，不踩珊瑚。
  • 费用：门票、ranger、港口费、浮潜装备是否包含必须书面确认。

住宿

维莫拉斯宾馆 / Wimolas Guesthouse

安全

风浪大时宁可少去点位，不要求船长硬赶。

当天晚餐

选择离酒店近的餐厅，早点休息，为第二天飞巴厘岛留状态。

2026-10-01D6
飞巴厘岛

Labuan Bajo → Bali Ngurah Rai → Badung / Kuta, Bali

从科莫多国际机场飞往巴厘岛伍拉·赖国际机场。抵达后前往 Badung / Kuta, Bali 区域入住 Rhadana Hotel。Badung 是巴厘岛南部主要旅游县，Kuta、Seminyak、Canggu、Jimbaran、Nusa Dua 多在这一带或相邻区域。当天尽量不安排远距离景点，把它作为洗衣、补给和海边散步日。

  • 机场到酒店：Kuta / Badung, Bali 视交通约 20-60 分钟；用 Grab / Gojek 或酒店接送。
  • 巴厘岛游客税：如果此前未在 Love Bali 支付，抵达巴厘岛后尽快完成。
  • 补给：换现金、买水果/水/防晒、确认 Nusa Penida 一日游接送时间。
  • 晚间：可去 Kuta / Seminyak 周边轻松吃饭，不建议熬夜。

住宿

拉达纳酒店 / Rhadana Hotel

当天策略

从“探险段”切到“海岛段”，恢复比打卡更重要。

备选

若航班早到，可加 Jimbaran 简单日落海鲜，把 D8 晚上压力降低。

2026-10-02D7
Nusa Penida

Klungkung / Nusa Penida 西线：Kelingking Beach → Broken Beach → Angel's Billabong

经典 Nusa Penida 西线。Nusa Penida 行政上属于 Klungkung / Nusa Penida，而不是巴厘岛本岛南部。早上从 Sanur 码头坐船到 Nusa Penida，岛上包车游览 Kelingking Beach、Broken Beach、Angel's Billabong，傍晚返程巴厘岛。

  • 码头：Sanur 码头需提前到，雨天和节假日排队更久；拖鞋不适合码头搬运行李，穿可湿鞋。
  • Kelingking：上方观景最稳；下海滩坡陡且回程耗体力，不建议在一日游里硬下到底。
  • Broken Beach：适合航拍/广角，注意悬崖边没有完整护栏。
  • Angel's Billabong：只看低潮和海况，涨潮或大浪时不要下水，不要站在外侧湿滑岩面。
  • 路况：岛上西线颠簸，晕车药和水必须带。

住宿

拉达纳酒店 / Rhadana Hotel

交通

酒店接送 → Sanur 船 → Nusa Penida 包车 → 回船 → 酒店。

备选

若海况差取消登岛，改 Uluwatu / Melasti Beach / Garuda Wisnu Kencana。

2026-10-03D8
北线到南线

Buleleng / Lovina Dolphins → Gianyar / Ubud Palace → Jimbaran Beach

这天跨度很大：从南部酒店凌晨出发去北部 Buleleng / Lovina 看海豚，再折返中部 Gianyar / Ubud，傍晚去 Jimbaran。建议只在你愿意接受“超早起 + 长车程”的情况下执行。

  • Lovina：通常 02:30-03:00 从 Kuta / Badung 出发，05:30 左右上船，海豚看到率高但不是保证。
  • 船上：穿防风外套，带防水袋；不要期待安静独享，旺季船较多。
  • Ubud Palace：停留 30-60 分钟即可，可顺路吃午餐和买咖啡。
  • Jimbaran：日落前 45 分钟到海滩，选择明码标价店，点餐前确认重量和服务费。
  • 现实提醒：从 Lovina 到 Ubud，再到 Jimbaran 车程长，堵车会吞掉体验。

住宿

拉达纳酒店 / Rhadana Hotel

优化建议

如果想更舒服，可取消 Ubud Palace，Lovina 后回酒店午睡，傍晚只去 Jimbaran。

备选

不追海豚时，改成 Ubud + Tegallalang + Jimbaran 的中南线。

2026-10-04D9
海神庙日落

Tanah Lot → 滑翔伞

海神庙 Tanah Lot 是巴厘岛经典海岸庙宇，日落剪影最好。当天另有滑翔伞安排，需提前确认实际基地、风向、保险、体重限制和取消政策。

  • Tanah Lot：涨潮时庙宇被海水包围，低潮可更接近外侧，但仍注意浪和湿滑礁石。
  • 拍照：日落前 60-90 分钟到，先找机位，再慢慢等光线。
  • 滑翔伞：巴厘岛常见基地在 Timbis / Nusa Dua / Uluwatu 一带，和 Tanah Lot 不在同一方向，需核实你行程里“克里达·曼达拉·伊·科图·洛特里”的具体坐标。
  • 最后一晚：整理行李、确认次日机场接送、把湿衣物分袋。

住宿

拉达纳酒店 / Rhadana Hotel

重点

把 Tanah Lot 放日落；滑翔伞放上午/中午依风况执行。

备选

滑翔伞取消时，改 Uluwatu Temple + Kecak Fire Dance 或 Canggu 咖啡海滩。

Bali Ngurah Rai → Singapore Transit → Beijing Daxing

伍拉·赖国际机场飞北京大兴，中转新加坡约 11 小时 45 分，跨天中转，行李直挂北京。若你持中国普通护照，目前中新互免签证安排允许短期入境新加坡最多 30 天，但仍需满足入境官判断、填写 SG Arrival Card，并携带后续机票。

  • 新加坡中转：若入境游玩，提前提交 SG Arrival Card，保存返程登机牌/行程单。
  • 11 小时玩法：轻装入境，选择 Jewel Changi、滨海湾短线或机场休息，不建议安排太满。
  • 跨天：确认登机口和日期，手机日历用当地时间，避免把 10 月 6 日航段看错。

住宿

无，跨天飞行。

D4 · 2026-09-29 · Labuan Bajo 海钓半日游

这天是恢复日，别把它塞满。目标是适应海况、补给、确认 D5 出海。

出发前
问船家六件事：几点接、哪里集合、船上有没有救生衣、是否含水/午餐、钓具是否包含、风浪大是否退款或改期。

海钓半日。感觉晕就看远处水平线，不低头刷手机。别靠船边站太久。
下午

D5 · 2026-09-30 · Komodo National Park

Pitravel 原数据为 13:00-15:30 科莫多国家公园，但一日出海通常更早。以船公司最终时间为准。

D9 · 2026-10-04 · Paragliding → Tanah Lot

涨潮时不要靠近浪打湿的礁石；低潮时也不要走到外侧湿滑区域。保管好拖鞋和相机。
晚上
回 Rhadana Hotel 整理返程行李。把托运行李和随身包分开：充电宝、证件、药品、外套必须随身。

Operational Roadbook

路书

这部分按“实际怎么走”写，适合出发当天照着核对。

D1

北京大兴 → 吉隆坡 → Surabaya 泗水 → Lumajang, East Java

值机时确认行李直挂 SUB、两段登机牌、吉隆坡是否需要换航站楼。KLIA 7 小时中转可吃饭、充电、补觉；若不入境，活动范围受航站楼限制。泗水落地后先过海关、取 SIM/现金，再联系司机。夜车去 Lumajang 以“安全抵达”为主，中途只停正规加油站/便利店。

地图搜索：Juanda Airport → Lumajang

D2

Lumajang, East Java → Tumpak Sewu → Cemoro Lawang / Cafe Lava Hostel

早出发去瀑布入口，先观景台，后谷底。瀑布结束后换干衣服再坐车，避免山路冷风。去 Bromo 山脚通常需经 Malang / Probolinggo 方向，路程长且山路弯多。抵达 Cafe Lava Hostel 后先确认 Jeep 司机、出发时间、是否含火山口和沙海。

地图搜索：Tumpak Sewu → Cafe Lava Hostel

D3

Bromo 日出 → Surabaya Airport → West Manggarai / Labuan Bajo

02:30 左右出门，日出后去沙海和火山口，预计 09:00-10:00 回住宿。洗漱、早餐、退房后立即去泗水机场。这个转场最怕堵车和火山段拖延，航班时间若偏早，应删减火山口或缩短停留。

地图搜索：Mount Bromo → Juanda Airport

D4-D5

West Manggarai / Labuan Bajo 海上两日

D4 半日海钓用于适应海况和恢复体力；D5 科莫多国家公园必须提前确认船公司资质、救生衣、午餐、浮潜装备、门票和 ranger。所有出海均以船长对风浪判断为准。Labuan Bajo 镇不大，酒店到码头通常车程不长，但清晨集合不要卡点。

地图搜索：Labuan Bajo Harbour / Komodo National Park

D6-D7

West Manggarai / Labuan Bajo → Bali → Sanur → Klungkung / Nusa Penida 西线

D6 飞巴厘岛后住 Badung / Kuta, Bali，D7 清晨去 Sanur 码头。Nusa Penida 一日游建议选择“船票 + 岛上包车 + 酒店接送”组合，少一点现场协调。西线景点之间路况颠，Kelingking 下海滩会显著拉长时间，若返程船固定，不建议下到底。

地图搜索：Sanur Harbour → Kelingking Beach

D8-D9

Buleleng / Lovina · Gianyar / Ubud · Jimbaran · Tanah Lot · Paragliding

D8 是全程第二高强度日：如果坚持 Buleleng / Lovina 海豚，从南部酒店往返会很累，司机要提前锁定。D9 滑翔伞需要按风况调整，不宜和 Tanah Lot 日落硬绑同一方向；把日落留给 Tanah Lot，滑翔伞以安全窗口优先。

地图搜索：Kuta → Lovina → Ubud → Jimbaran

D10

Bali → Singapore → Beijing

离开巴厘岛前检查是否有未干衣物、刀具/液体是否托运、充电宝随身。新加坡中转若入境，提前填 SG Arrival Card；若只在机场内休息，确认下一程登机口开放时间。跨天航班把手机时区设为自动。

地图搜索：Ngurah Rai International Airport

Quick Checklist

出发前核对

提前 14-7 天

  • 申请印尼 eVOA / Visitor Visa。
  • 确认所有航班行李额和无餐食航段。
  • 锁定泗水接机、Bromo Jeep、Komodo 船、Nusa Penida 一日游、Lovina 包车。
  • 确认境外旅行保险覆盖火山、涉水、出海、滑翔伞等项目。

提前 72 小时

  • 填写 All Indonesia Arrival Card。
  • 支付 Love Bali Levy 或确认抵达后支付方式。
  • 打印护照首页、签证、酒店、保险、返程机票。
  • 再次确认 Komodo 船和 Nusa Penida 船是否受天气影响。

每天晚上

  • 把第二天现金、小票、证件、药品放到同一随身袋。
  • 充满手机、相机、充电宝。
  • 离线保存司机电话、集合点、酒店地址。
  • 看天气、潮汐和风浪，必要时删减景点。
